# CampusQuiet

1.Overview
CampusQuiet is an Android application designed to help maintain silence in campus environments by controlling mobile device behavior using Bluetooth-based interaction.

This project emphasizes simplicity, usability, and hands-on learning of core Android development concepts.

----------------------------------------------------------------------------------------------

2.Key Features
- Simple and user-friendly interface  
- Bluetooth-based device communication  
- Proper handling of Android runtime permissions  
- Lightweight and efficient application  

---------------------------------------------------------------------------------------------

3.Technology Stack
- **Programming Language:** Kotlin  
- **Development Environment:** Android Studio  
- **Platform:** Android  
- **Build System:** Gradle  

-----------------------------------------------------------------------------------------------

4.Getting Started
Follow the steps below to run the project locally:

1. Clone the repository  
2. Open the project in **Android Studio**  
3. Allow Gradle to complete syncing  
4. Run the app on an Android emulator or a physical device  

----------------------------------------------------------------------------------------------

5.Project Notes
- Developed primarily for learning and academic purposes  
- Follows standard Android development practices  
- Suitable for beginner-level understanding and experimentation
